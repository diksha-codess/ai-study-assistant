# 🌱 AI Study Assistant

## 📌 Problem Statement

Students struggle to plan their studies and manage tasks effectively.

## 💡 Solution

This project generates a 5-day study plan based on any topic.

## 🚀 Features

* Auto study plan generator
* Simple UI
* Fast response
* Open source

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Python (Flask)

## ▶️ How to Run

1. Install dependencies
2. Run backend
3. Open frontend
